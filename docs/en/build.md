# Build

## Local Build

The repository includes a [Makefile](https://github.com/touero/easier-docker/blob/master/Makefile) for common build steps.

```shell
make
```

## Development Install

```shell
pip install -e .[dev]
```

## Run Tests

```shell
pytest
```
